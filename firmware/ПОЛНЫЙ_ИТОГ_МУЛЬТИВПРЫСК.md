## 🎓 Ключевые выводы

### Система координат RUSEFI

```
Таблица injectionPhase: BTDC (опережение)
↓
Автоматически преобразуется в абсолютный угол при расчете
```

```
Таблица secondInjectionAngleTable: BTDC (опережение)
↓
НУЖНО вручную преобразовать: baseAngle = 720 - btdcAngle
↓
Потом используется как обычный абсолютный угол
```

### Длительность впрыска

**Правильный способ:**
```
1. Получить массу: injectionMass[cyl]
2. Разделить: pulseMass = mass * (ratio / 100)
3. Рассчитать: pulseDuration = injectorModel(pulseMass)
   ↑ Это учитывает dead time каждого импульса!
```
